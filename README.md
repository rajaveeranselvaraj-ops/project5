This project focuses on healthcare data analysis using Python, Pandas, NumPy, and Matplotlib. A synthetic dataset containing 10,000 patient records was generated to study different health parameters and identify patient risk levels. The dataset includes details such as patient ID, age, gender, blood pressure, sugar level, cholesterol, and heart rate. Random values were generated using NumPy to simulate realistic healthcare data for analysis and visualization purposes.
The project begins with data creation and preprocessing. Patients were categorized based on their heart rate into “Normal” and “High” heart rate status. A risk analysis system was also implemented to classify patients into “High Risk” and “Low Risk” categories depending on medical conditions such as high blood pressure, high sugar level, high cholesterol, or increased heart rate. Additionally, patients were grouped into different age categories such as Young, Middle, and Senior to make the analysis more meaningful.
Several statistical analyses were performed to understand the dataset better. Average values of health parameters were calculated, and gender distribution along with risk distribution was analyzed. Different visualizations were created using Matplotlib, including bar charts, pie charts, histograms, scatter plots, boxplots, and correlation heatmaps. These graphs helped in understanding relationships between medical attributes such as age and blood pressure, as well as identifying patterns in cholesterol and heart rate data.
The analysis showed that a large number of patients fell into the high-risk category due to abnormal medical conditions. The gender distribution was almost equal between male and female patients. Correlation analysis also provided useful insights into how different health factors are related to one another. Overall, this project demonstrates how healthcare data can be generated, processed, analyzed, and visualized effectively using Python-based data science tools.
